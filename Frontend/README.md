# ConectaSV Frontend

Frontend de la aplicación ConectaSV, plataforma de conexión laboral.

## Tecnologías

- HTML5
- CSS3
- JavaScript (Vanilla)

## Estructura

- `index.html` - Página principal
- `admin/` - Panel de administrador
- `Candidate/` - Panel de candidato
- `Employee/` - Panel de empleador
- `css/` - Estilos CSS
- `js/` - Scripts JavaScript

## Desarrollo

Simplemente abre `index.html` en un navegador web.

Para desarrollo con servidor local, puedes usar un servidor simple como:
```bash
npx serve .
```

## API

El frontend se conecta a la API del backend en las rutas definidas.
